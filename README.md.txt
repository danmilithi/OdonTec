# OdonTec

## Status do projeto
Em desenvolvimento

## Tecnologias
- Java (Swing)
- MySQL
- JDBC

## Time de desenvolvedores
- Daniel Viana

## Objetivo do software
O OdonTec é um sistema desktop para gestão de clínicas odontológicas, incluindo cadastro de pacientes, dentistas, produtos, fornecedores e agendamento de tratamentos.

## Funcionalidades
- Cadastro de pacientes, dentistas, produtos e fornecedores
- Agendamento de tratamentos
- Autenticação de usuários com permissões (Gerente, Atendente, Financeiro, Dentista)
- Visualização de registros e relatórios
- Controle de estoque de produtos
